## For this application you just need to have the app password for you google account. Below is the process you can follow-

1.Turn on 2-Step Verification: Go to your Google Account settings, select Security, locate 2-Step Verification, and turn it on (App Passwords will not work without this).

2.Go to App Passwords(Search "App Password" in the search of the previous page): Navigate directly to the Google App Passwords Page.

3.Name the App: In the App name text box, type a clear name like Python Automation.

4.Generate: Click the Create button.

5.Copy Code: Copy the 16-character code displayed inside the pop-up window immediately.

6.Finalize: Click Done (remember to remove all spaces when pasting this code into your .env file).
